# 📚 Glossário

Para facilitar o entendimento, aqui estão as definições dos termos usados neste projeto:

| Termo                     | Definição                                                                                                                              |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| **Slug**                  | Uma versão simplificada do nome, como um ID, usada para comandos. Não contém espaços ou acentos. (Ex: `Dragão Raro` -> `dragao_raro`). |
| **Spawn**                 | O evento de "nascer" ou aparecer um pacote no chat da live.                                                                            |
| **Claim**                 | O ato de resgatar ou reivindicar um pacote que apareceu via spawn.                                                                     |
| **Overlay**               | A camada visual transparente que fica em cima da sua imagem da stream no OBS/Streamlabs.                                               |
| **Tipagem (Tipo)**        | A categoria ou elemento que classifica uma carta, permitindo organizar coleções e filtrar cartas por temas específicos.                |
| **Raridade**              | O nível de escassez de uma carta, que define a chance de ela ser sorteada.                                                             |
| **Cooldown**              | O tempo de espera obrigatório entre o uso de um comando e outro para evitar spam.                                                      |
| **Inventário**            | O conjunto de cartas e pacotes que um usuário possui.                                                                                  |
| **OAuth Token**           | Uma chave secreta que dá permissão para o bot agir e falar em nome da conta no chat da Twitch.                                         |
| **Aplicação Dev Twitch**  | A API moderna da Twitch usada para buscar dados como fotos de perfil e status da live.                                                 |
| **Client ID / Secret**    | Credenciais geradas no console de desenvolvedor da Twitch para permitir que o bot se comunique com a Helix API.                        |
| **TMI (IRC)**             | A tecnologia de chat da Twitch que o bot usa para ler e enviar mensagens em tempo real.                                                |
| **Evento Sorteio/Raffle** | Tipo de spawn onde o vencedor é escolhido aleatoriamente entre todos que participaram.                                                 |
| **Event Rápido/Fastest**  | Tipo de spawn onde ganha quem digitar o comando primeiro (ou os X primeiros).                                                          |
| **XP (Experiência)**      | Pontos ganhos por atividade, usados para subir o Nível (Level) do espectador no sistema.                                               |
| **Painel Administrativo** | A interface no seu computador onde você gerencia cartas, usuários e configurações do bot.                                              |
