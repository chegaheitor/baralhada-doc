# 👥 Comunidade

Esta categoria é focada nos seus usuários. Aqui você gerencia a progressão, a economia pessoal de cada jogador e as interações entre eles, como trocas e ganhos de níveis.

#### 🤝 O que você encontra aqui:

1. **Gerenciar Usuários**: Visualize o inventário de qualquer jogador, adicione/remova itens e gerencie banimentos se necessário.
2. **Sistema de Pontos**: Configure a base da sua economia. Defina quantos pontos o chat ganha por mensagem enviada ou por tempo assistido.
3. **Trocas Manuais**: Intervenha quando necessário. Realize trocas diretas entre usuários ou ajude a resolver pendências de negociações.
4. **Níveis e XP**: Configure a curva de progressão. Defina quanto XP é necessário para subir de nível e quais as recompensas por atingir novos patamares.
5. **Personalização**: Dê a cara da sua live ao bot. Mude as cores das raridades, os nomes dos tipos de cartas e muito mais.

{% hint style="warning" %}
Sempre verifique os **Usuários** mais ricos no painel. Se alguém subir de nível rápido demais, pode ser hora de ajustar sua curva de **XP**.
{% endhint %}

{% hint style="info" %}
**Dica de Streamer:** Use o sistema de **Níveis** para dar cargos especiais no seu Discord ou recompensas exclusivas na live. Isso cria um senso de progressão real além das cartas!
{% endhint %}
