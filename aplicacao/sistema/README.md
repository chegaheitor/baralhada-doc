# Sistema

A base tecnológica do seu bot. Nesta categoria você configura a "espinha dorsal" do aplicativo, desde as mensagens que o bot envia até a segurança dos seus dados.

#### 🛡️ O que você encontra aqui:

1. **Gerenciar Comandos**: Ative, desative ou mude os nomes dos comandos que o chat usa (!abrir, !mochila, etc).
2. **Personalizar Mensagens**: Edite cada resposta do bot. Deixe o bot com a linguagem da sua comunidade usando variáveis dinâmicas.
3. **Segurança e Backups**: Proteja o progresso de todos. Configure faxinas no banco de dados e garanta que ninguém perca sua coleção.
4. **Overlay e Alertas**: O visual que aparece na live. Configure alertas de spawn, abertura de pacotes e trocas que saltam na tela do OBS.
5. **Configurações Gerais**: Onde tudo começa. Configure suas chaves de API da Twitch, porta do servidor e bot tokens.

***

{% hint style="info" %}
**Visual:** Use o **Overlay** em modo local para garantir zero atraso (delay) entre a ação no chat e a animação na tela.
{% endhint %}

{% hint style="danger" %}
**Segurança:** Nunca compartilhe sua tela enquanto estiver na página de **Configurações Gerais**. Seus tokens de acesso (OAuth) são privados e dão controle sobre a conta do seu bot!
{% endhint %}
