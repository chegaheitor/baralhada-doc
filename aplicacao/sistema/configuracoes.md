# Configurações

## Credenciais do Twitch

| Campo                    | Descrição                                                  |
| ------------------------ | ---------------------------------------------------------- |
| **Usuário do Bot**       | Username da conta bot (ex: `chegabotTCG`)                  |
| **Token OAuth**          | Token de autenticação gerado em https://twitchapps.com/tmi |
| **Nome do Canal**        | Seu canal (sem `@` ou `#`)                                 |
| **Twitch Client ID**     | Da aplicação em https://dev.twitch.tv                      |
| **Twitch Client Secret** | Da aplicação em https://dev.twitch.tv                      |

{% hint style="danger" %}
⚠️ Nunca compartilhe o token OAuth nem o Client Secret.
{% endhint %}

## Configurações do Bot

| Campo                           | Descrição                                                       |
| ------------------------------- | --------------------------------------------------------------- |
| **Duração do Overlay de Spawn** | Segundos que a animação de spawn fica na tela (via overlay OBS) |
| **Auto-Start do Bot**           | Conecta o bot automaticamente ao abrir o programa               |

## Configurações do Servidor

| Campo                 | Descrição                                    |
| --------------------- | -------------------------------------------- |
| **Porta do Servidor** | Porta HTTP local do backend (padrão: `3000`) |

{% hint style="warning" %}
⚠️ Mudar a porta exige reinicialização do aplicativo.
{% endhint %}

## Salvando Configurações

Clique em **Salvar Configurações**. Se a porta foi alterada, o sistema perguntará se deseja reiniciar agora.

## Controle do Bot

Na parte superior da tela de Configurações há o painel de status do bot:

* **Iniciar Bot** — conecta ao chat do Twitch
* **Parar Bot** — desconecta do chat

## Sobre / Documentação

No rodapé da página há o card de informações do **Baralhada Bot** com links para:
