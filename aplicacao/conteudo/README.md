# Conteúdo

Aqui é onde a mágica acontece. Esta categoria engloba todos os elementos que compõem o ecossistema de jogo do seu bot, desde as cartas individuais até os grandes eventos de spawn.

#### 🍱 O que você encontra aqui:

1. **Pacotes**: Crie e gerencie os conjuntos de cartas que seus usuários poderão abrir. Defina as chances de cada raridade e o visual dos pacotes.
2. **Cartas**: O coração da coleção. Gerencie cada carta individualmente, adicione imagens, nomes e vincule-as aos seus respectivos álbuns.
3. **Spawns e Eventos**: Controle o ritmo da live. Configure o tempo entre aparições automáticas de pacotes selvagens para o chat resgatar.
4. **Loja do Bot**: Configure a economia direta. Decida quais pacotes estão à venda, por quanto e se os usuários podem vender cartas de volta para o sistema.
5. **Sistema de Missões**: Crie objetivos dinâmicos para manter o chat engajado em troca de recompensas em pontos ou XP.
6. **Conquistas (Achievements)**: Recompense a fidelidade. Configure marcos que dão emblemas e status aos seus melhores colecionadores.

{% hint style="info" %}
**Dica de Streamer:** A variedade é a chave do engajamento! Tente lançar um novo **Pacote** temático a cada mês para manter o interesse dos seus colecionadores sempre renovado.
{% endhint %}

{% hint style="success" %}
As **Missões** são a melhor maneira de converter um "lurker" em um jogador ativo. Recompensas de login diário ou metas de mensagens funcionam muito bem!
{% endhint %}
