# Conquistas

## 🏆 O que são Conquistas

As conquistas são o "Hall da Fama" dos seus espectadores. Diferente das missões, elas celebram marcos de longo prazo e dão prestígio visual na live.

## Criando uma Conquista

{% stepper %}
{% step %}
### Vá em Painel → Conquista

Tenha acesso a lista de conquistas criadas
{% endstep %}

{% step %}
### Criar nova conquista

Clique em **Nova Conquista**
{% endstep %}

{% step %}
### Configurar a Conquista

Preencha os campos abaixo:

| Campo                     | Descrição                                              |
| ------------------------- | ------------------------------------------------------ |
| **Nome**                  | Título da missão                                       |
| **Descrição**             | Texto explicativo                                      |
| **Objetivo da Conquista** | Tipo de objetivo (ver tabela abaixo)                   |
| **Meta Quantitativa**     | Valor alvo (ex: 5 para "abrir 5 pacotes")              |
| **Identidade Visual**     | Uma imagem personalizada ou um símbolo de uma galeria. |
{% endstep %}
{% endstepper %}

## 🛠️ Tipos de Conquistas Disponíveis

| ID                    | Nome                       | Descrição                        |
| --------------------- | -------------------------- | -------------------------------- |
| `cards_total`         | Total de Cartas na Mochila | Ter X cartas no total            |

| `points_total`        | Atingir Pontuação          | Chegar a X pontos de saldo       |

| `trades_total`        | Total de Trocas            | Realizar X trocas                |

| `packs_opened`        | Total de Pacotes Abertos   | Abrir X pacotes                  |
| `buy_total`           | Total de Compras na Loja   | Comprar X itens                  |
| `buy_packs`           | Total de Pacotes Comprados | Comprar X pacotes                |
| `buy_cards`           | Total de Cartas Compradas  | Comprar X cartas                 |
| `sell_total`          | Total de Vendas na Loja    | Vender X itens                   |

## 🎖️ Como são Desbloqueadas

O bot monitora passivamente as ações dos usuários. Assim que um critério é atingido, a conquista é liberada.

### Exemplos de Critérios:

* **Colecionador**: Completar uma coleção específica 100%.
* **Magnata**: Acumular 1.000.000 de pontos na carteira.
* **Veterano**: Atingir o nível máximo (Nível 20).
* **Sorte Grande**: Tirar uma carta lendária específica.

## 💎 Valor Social e Retenção

* **Impacto de Prestígio**: Viewers que possuem conquistas de "Platina" ou "Ouro" são reconhecidos como os membros mais engajados da comunidade.
* **Vantagem**: Diferente de pontos que são gastos, as conquistas são **permanentes**, servindo como um currículo do espectador no seu canal.

{% hint style="info" %}
**Dica Visual**: Use ícones que combinem com a temática do seu canal! Se sua live é sobre piratas, use ícones de baús, bússolas e moedas de ouro para as conquistas.
{% endhint %}

{% hint style="warning" %}
Algumas conquistas complexas (como coleção completa) só são checadas quando o usuário interage com uma ação relacionada (ex: ganha uma carta).
{% endhint %}
