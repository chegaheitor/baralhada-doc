# Spawn Pacote

## ⌨️ Como usar

```
!spawnpack <id_pacote>
```

## 📝 Descrição

Força o lançamento imediato de um pacote no chat. Ignora os cronômetros automáticos e gera o alerta visual na mesma hora.

## 📖 Exemplo de Resposta

> 📦 @moderador disparou um spawn manual de **Pacote Especial**!
