# Administração

## 👮 Comandos: Administração

Esta categoria é exclusiva para o **Streamer** e seus **Moderadores**. Estes comandos permitem o controle total sobre a economia e os usuários da live.

### ⚙️ Controle de Gestão

Os administradores podem usar estes comandos para:

* **Manutenção de Saldo**: Corrigir erros de pontuação ou premiar viewers manualmente.
* **Gestão de Inventário**: Dar itens promocionais ou remover itens em casos de disputa.
* **Segurança**: Bloquear usuários que estejam abusando do sistema ou perturbando o chat.
* **Eventos**: Forçar o aparecimento de pacotes para momentos especiais da live.

***

#### ⌨️ Comandos Disponíveis

* **`!dar`** / **`!remover`**: Adição ou remoção forçada de itens.
* **`!addpontos`** / **`!rempontos`**: Gestão manual de saldo de pontos.
* **`!spawnpack`**: Execução manual de um drop de pacote.
* **`!bloquear`** / **`!desbloquear`**: Controle de acesso ao bot.

{% hint style="danger" %}
Use os comandos de `!dar` e `!addpontos` com sabedoria para evitar que a economia do seu canal perca o valor e o desafio!
{% endhint %}
