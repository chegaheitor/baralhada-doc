# 🤝 Troca e Presentes

## 🤝 Comandos: Trocas e Presentes

Esta categoria é dedicada à interação direta entre os membros da comunidade. Aqui, o foco sai da relação "Jogador vs Sistema" e passa para o social, permitindo que os viewers colaborem para completar seus álbuns.

### 🤝 Colaboração e Comunidade

O sistema de trocas e presentes é fundamental para:

* **Completar Coleções**: Conseguir aquela carta rara que você não deu sorte de tirar no pacote.
* **Interação Social**: Criar laços entre os membros mais ativos do chat.
* **Generosidade**: Ajudar novos viewers que acabaram de chegar na live com cartas básicas.

***

#### ⌨️ Comandos Disponíveis

* **`!troca`**: Inicia uma proposta formal de negociação com outro usuário.
* **`!trocaoferta`**: Especifica quais cartas você está disposto a dar na troca.
* **`!trocasim`** / **`!trocanao`**: Aceita ou recusa a proposta final de troca.
* **`!presentear`**: Envia um item diretamente para outro usuário sem contrapartida.

{% hint style="info" %}
O sistema de **Troca** possui travas de segurança que impedem que um usuário "roube" outro. A transação só ocorre quando **ambos** os lados confirmam a oferta final.
{% endhint %}

{% hint style="info" %}
Use o comando `!presentear` para fazer sorteios rápidos ou premiar viewers que acertaram uma pergunta difícil na sua live!
{% endhint %}
