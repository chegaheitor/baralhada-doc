# 🎁 Presente

## ⌨️ Como usar

```
!presentear @usuario <id_item>
```

## 📝 Descrição

Seja generoso e ajude um amigo! Este comando permite enviar uma carta ou um pacote do seu inventário para outro espectador de forma direta.

## 📖 Exemplo de Resposta

> 🎁 @usuario presenteou @amigo com o item **Dragão de Fogo**!

<details>

<summary>Mensagens Relacionadas</summary>

| Chave                   | Quando dispara                                                    |
| ----------------------- | ----------------------------------------------------------------- |
| `gift_success`          | Quando o presente é enviado com sucesso para o destinatário.      |
| `gift_received`         | Mensagem enviada para o destinatário avisando sobre o presente.   |
| `trade_error_not_found` | Quando o usuário marcado não foi encontrado ou não existe no bot. |
| `trade_error_item`      | Quando você não possui o item que está tentando dar de presente.  |

</details>

{% hint style="info" %}
Presentear outros usuários é uma ótima forma de ganhar XP extra e desbloquear conquistas secretas de "Doador".
{% endhint %}
