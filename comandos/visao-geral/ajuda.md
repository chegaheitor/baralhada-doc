# Ajuda

## ⌨️ Como usar

```
!ajuda
```

## 📝 Descrição

Envia uma mensagem no chat com os principais comandos disponíveis e uma breve descrição de cada um. Útil para novos viewers que não conhecem o sistema.

## 📖 Exemplo de Resposta

> 🃏 Baralhada Bot | Comandos: !claim (pegar spawn) | !open (abrir pacote) | !deck (suas cartas) | !bags (seus pacotes) | !points (seus pontos) | !buy (comprar) | !sell (vender) | !daily (recompensa diária) | !trade @user (trocar)



<details>

</details>

Edite a mensagem `help_message` no **Painel → Mensagens** para:

* Incluir apenas os comandos ativos no seu canal
* Usar o visual/tom de mensagem do seu canal
{% endhint %}
