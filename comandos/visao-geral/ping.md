# Ping

## ⌨️ Como usar

```
!ping
```

## 📝 Descrição

Verifica rapidamente se o bot está online e respondendo ao chat. Útil para confirmar que o sistema está funcionando antes de uma live.

## 📖 Exemplo de Resposta

> 🏓 Pong! Bot online e funcionando!

<details>

<summary>Mensagens Relacionadas</summary>

| Chave           | Quando dispara   |
| --------------- | ---------------- |
| `ping_response` | Resposta ao ping |

</details>

{% hint style="info" %}
Edite a mensagem `ping_response` no **Painel → Mensagens** para:

* Incluir apenas os comandos ativos no seu canal
* Usar o visual/tom de mensagem do seu canal
{% endhint %}
