# Deck

## ⌨️ Como usar

```
!deck
```

## 📝 Descrição

Exibe um resumo da sua coleção de cartas atual e o total de itens que você já coletou.

## 📖 Exemplo de Resposta

> 🃏 @usuario, você possui 42 cartas no total. Algumas delas: Dragão (R), Espada (C), Mago (I), Escudo (C), Poção (C)

<details>

<summary>Mensagens Relacionadas</summary>

| Chave       | Quando dispara                                                          |
| ----------- | ----------------------------------------------------------------------- |
| `deck_info` | Exibe o total de cartas e uma seleção aleatória de 5 itens do seu deck. |

</details>

{% hint style="info" %}
Se quiser ver o progresso detalhado de uma coleção específica, use o comando `!album <nome_colecao>`.
{% endhint %}
