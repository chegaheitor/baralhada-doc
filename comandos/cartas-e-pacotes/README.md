# 📦 Cartas e Pacotes

## 🎴 Comandos: Cartas e Pacotes

Esta é a categoria principal do jogo. Aqui estão os comandos que lidam diretamente com o inventário, a coleção e a abertura dos itens colecionáveis.

### 🃏 O Ciclo do Colecionador

Nesta seção, detalhamos como o viewer interage com os itens:

1. **Coleta**: Como reagir a um spawn e coletar o pacote (`!resgatar`).
2. **Revelação**: O momento de abrir o pacote e ver as novas cartas.
3. **Organização**: Como visualizar sua "mochila" de pacotes e seu "deck" de cartas.
4. **Exibição**: Como abrir o álbum visual para ver o que falta.

***

#### ⌨️ Comandos Disponíveis

* **`!resgatar`**: Resgate de drops no chat.
* **`!abrir`**: Abertura de pacotes e sorteio de cartas.
* **`!mochila`**: Inventário de pacotes lacrados.
* **`!deck`**: Inventário de cartas colecionadas.
* **`!carta`** / **`!pacote`**: Consulta de detalhes técnicos de um item.
* **`!album`**: Visualização visual da coleção completa.
* **`!spawn`**: Consulta de status de drops e tempos.

{% hint style="info" %}
Lembre seus viewers que pacotes na **Mochila** não dão cartas automaticamente; eles **precisam** usar o comando `!abrir` para que as cartas entrem no **Deck**.
{% endhint %}
