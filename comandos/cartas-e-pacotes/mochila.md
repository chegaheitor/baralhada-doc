# Mochila

## ⌨️ Como usar

```
!mochila
```

## 📝 Descrição

Lista todos os pacotes de cartas que você possui no inventário e que ainda não foram abertos.

## 📖 Exemplo de Resposta

> 🎒 @usuario, sua mochila contém: 📦 Pacote Básico (basico) \[x2], 📦 Pacote Lendário (lendario) \[x1]

<details>

<summary>Mensagens Relacionadas</summary>

| Chave       | Quando dispara                                          |
| ----------- | ------------------------------------------------------- |
| `bag_list`  | Exibe a lista de pacotes que o usuário possui.          |
| `bag_empty` | Quando você usa o comando mas não possui nenhum pacote. |

</details>

{% hint style="info" %}
Use o comando `!abrir <id_do_pacote>` para abrir os pacotes listados aqui.
{% endhint %}
