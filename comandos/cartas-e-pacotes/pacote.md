# 🎁 Pacote

## ⌨️ Como usar

```
!pacote <id_pacote>
```

## 📝 Descrição

Veja as informações técnicas de um pacote de cartas, como as chances de drop de cada raridade e o preço de venda/compra.

## 📖 Exemplo de Resposta

> 📦 Detalhes do Pacote: **Básico** | Resgate: Rápido | Chances: C: 60%, I: 25%, R: 10%, E: 4%, L: 1%

<details>

<summary>Mensagens Relacionadas</summary>

| Chave                       | Quando dispara                                            |
| --------------------------- | --------------------------------------------------------- |
| `pack_info`                 | Exibe as estatísticas e informações do pacote solicitado. |
| `error_pack_info_not_found` | Se o ID do pacote digitado não for encontrado no sistema. |

</details>

{% hint style="info" %}
Além da resposta no chat, este comando faz o pacote aparecer no **Overlay** da live!
{% endhint %}
