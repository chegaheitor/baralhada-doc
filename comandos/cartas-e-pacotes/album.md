# Album

## ⌨️ Como usar

```
!album <id_pacote>
```

## 📝 Descrição

Exiba seu progresso visual em uma coleção diretamente na tela da live (Overlay). Útil para mostrar quais cartas você já tem e quais faltam para trocar.

## 📖 Exemplo de Resposta

> 📖 @usuario, seu progresso no álbum **Básico** está sendo exibido na tela agora!

<details>

<summary>Mensagens Relacionadas</summary>

| Chave                    | Quando dispara                                                     |
| ------------------------ | ------------------------------------------------------------------ |
| `album_overlay_notified` | Confirma no chat que o álbum apareceu no overlay para todos verem. |
| `error_pack_not_found`   | Se o ID do pacote digitado não existir no sistema.                 |

</details>

{% hint style="info" %}
O álbum aparece apenas por alguns segundos no **Overlay**.&#x20;
{% endhint %}
