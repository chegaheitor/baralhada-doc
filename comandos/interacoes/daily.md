# ☀️ Daily

## ⌨️ Como usar

```
!daily
```

## 📝 Descrição

Sua recompensa gratuita diária. Permite resgatar uma quantia de pontos uma vez a cada 24 horas para ajudar na sua economia.

## 📖 Exemplo de Resposta

> 📅 @usuario, você resgatou sua recompensa diária de 500 pontos! Seu saldo atual: 2.150 pontos.

<details>

<summary>Mensagens Relacionadas</summary>

| Chave            | Quando dispara                                                   |
| ---------------- | ---------------------------------------------------------------- |
| `daily_success`  | Quando você resgata os pontos com sucesso.                       |
| `daily_cooldown` | Quando você tenta usar o comando antes de completar as 24 horas. |

</details>

{% hint style="info" %}
O valor do seu `!daily` aumenta conforme o seu **Nível (Level)**! Quanto mais você sobe de nível, maior é a sua recompensa diária.
{% endhint %}
