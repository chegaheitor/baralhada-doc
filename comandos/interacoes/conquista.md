# 🏆 Conquista

## ⌨️ Como usar

```
!conquista
```

## 📝 Descrição

Consulte sua galeria de troféus. Exibe o resumo de quantas conquistas você já desbloqueou no sistema.

## 📖 Exemplo de Resposta

> 🏆 @usuario, você desbloqueou 12 de 50 conquistas! | Última: Colecionador de Elite - Abriu 100 pacotes.

<details>

<summary>Mensagens Relacionadas</summary>

| Chave                        | Quando dispara                                               |
| ---------------------------- | ------------------------------------------------------------ |
| `achievement_summary`        | Exibe o total de conquistas e a última desbloqueada.         |
| `no_achievements_configured` | Se o streamer ainda não configurou nenhuma conquista no bot. |

</details>
