# Pontos

## ⌨️ Como usar

```
!pontos
```

## 📝 Descrição

Verifica o seu saldo de pontos atual. Os pontos são usados para comprar cartas ou pacotes na loja.

## 📖 Exemplo de Resposta

> 💰 @usuario, seu saldo atual é de 1.500 pontos.

<details>

<summary>Mensagens Relacionadas</summary>

| Chave            | Quando dispara                            |
| ---------------- | ----------------------------------------- |
| `points_balance` | Exibe o saldo atual de pontos do usuário. |

</details>

{% hint style="info" %}
Os viewers ganham pontos automaticamente apenas por conversar no chat da live!
{% endhint %}

## 💰 Como Ganhar Pontos

* Ficando na live (pontos por tempo, configurável)
* Mandando mensagem no chat
* Usando `!daily`
* Vendendo itens com `!vender`&#x20;
* Completando missões e conquistas

## 💸 Como Gastar Pontos

* Comprando cartas e pacotes com `!comprar`
