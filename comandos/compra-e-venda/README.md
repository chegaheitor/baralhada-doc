# Compra e Venda

## 💰 Comandos: Compra e Venda

Esta categoria lida com a movimentação de itens e o mercado interno do canal. São os comandos que permitem que as cartas circulem e façam a economia girar.

### 🛒 A Economia em Movimento

Os viewers podem gerenciar sua coleção de três formas principais:

* **Mercado (Loja)**: Comprar itens novos diretamente do sistema ou vender repetidas para o bot.

***

#### ⌨️ Comandos Disponíveis

* **`!comprar`**: Aquisição de itens na loja oficial.
* **`!vender`**: Transformar itens em pontos novamente.

{% hint style="info" %}
O sistema de **Troca** é o mais seguro, pois garante que ambos os usuários confirmem a oferta antes que qualquer item mude de dono.
{% endhint %}
